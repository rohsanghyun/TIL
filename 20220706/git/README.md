# git 정리



- 내용

  - github username 변경 방법.

    - https://github.com/settings/admin

      

  - github원격 저장소 만들기
    - repo이름, 저장소설명, 공개여부 확인
    
  - 로컬에 있는 폴더 github로 push 하는 방법.
    - 원격저장소 추가해 으로 깃허브 저장소이름
    - git remote -v
    - git push <원격저장소이름> <브랜치이름>



> 깃허브 아무거나 누르지 말기
>
> 내일은 pull 진행 예정
